# Rellax-Parallax

## Installation

```yarn add react-rellax````

## Add Parallax

```<div
  style={{
    height: '100vh',
    background: 'red',
  }}
>
  <Parallax
    style={{
      height: '100vh',
      display: 'flex',
      alignItems: 'center',
      justifyContent: 'center',
    }}
    data-rellax-speed="-4"
  >
    Hello World
  </Parallax>
</div>```
